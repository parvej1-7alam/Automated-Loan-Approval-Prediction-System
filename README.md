##  **Project Overview**

Financial institutions receive thousands of loan applications every day.  
Manually evaluating them is **slow**, **inconsistent**, and prone to **human bias**.

This project builds a **machine learning‚Äìbased loan approval prediction system** to automate the decision-making process. The model predicts whether a loan should be **Approved** or **Rejected** using demographic, financial, and credit parameters.

---

##  **Objectives**

-  **Increase efficiency** of loan processing  
-  **Improve prediction accuracy** using ML  
-  **Enhance fairness** with data-driven insights  
-  **Identify key factors** influencing approval decisions  

---

##  **Key Features**

- Complete **data loading, cleaning, and preprocessing**
- Exploratory Data Analysis (**EDA**) with visual insights  
- Feature engineering & handling missing values  
- Multiple ML models tested (Logistic Regression, Decision Tree, Random Forest, etc.)  
- Final model evaluation using Accuracy, Precision, Recall, F1-Score, and Confusion Matrix  
- Clean, well-structured Jupyter Notebook workflow  

---

##  **Dataset Information**

The dataset includes common applicant parameters:

- Applicant Income  
- Co-Applicant Income  
- Loan Amount & Duration  
- Credit History  
- Gender, Marital Status, Dependents  
- Education Level  
- Property Area  
- Loan Approval Status (Target)
