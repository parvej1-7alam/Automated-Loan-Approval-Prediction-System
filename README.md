# Automated-Loan-Approval-Prediction-System
Repository for a Logistic Regression model automating loan decisions. Primary goal: replace manual assessment with a reliable, scalable ML solution.  Key Results: Accuracy 81.5%; High F1-Score (0.852) for Approved class. Precision (0.849) reduces False Approvals.  Success: Increased Efficiency (instant decisions), Reduced Risk, Enhanced Fairness.
